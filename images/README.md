# Images Folder

Add your project thumbnail images here. The site is configured to use 6 project thumbnails.

## Required Images:
You need to add 6 thumbnail images from your Instagram/TikTok content:

1. **project1-thumb.jpg** - Instagram Reel (https://www.instagram.com/reel/DHBWYqbMLMK/)
2. **project2-thumb.jpg** - Creative Content (https://www.instagram.com/reel/DEkLYkBpfUV/)
3. **project3-thumb.jpg** - Production & Edit (https://www.instagram.com/reel/DFtAvD1p38y/)
4. **project4-thumb.jpg** - Community Event (https://www.instagram.com/reel/CzPvjSEOeTs/)
5. **project5-thumb.jpg** - TikTok Campaign (https://vt.tiktok.com/ZSa8PQkQ5/)
6. **project6-thumb.jpg** - Viral Content (https://vt.tiktok.com/ZSa8fcmNP/)

## How to Get Thumbnails:
1. Visit each Instagram/TikTok link above
2. Take a screenshot or download the video thumbnail
3. Save as the corresponding filename (project1-thumb.jpg, project2-thumb.jpg, etc.)
4. Place all 6 images in this folder

## Image Specifications:
- **Format**: JPG or PNG
- **Recommended Size**: 800x1000px (portrait orientation)
- These thumbnails will be used in both the Projects section AND the landing page background effect

After adding images, update the `src` attributes in index.html to point to your local images:
```html
<img src="images/project-01.jpg" alt="Project 1" class="project-image">
```
