# Jasnoor Saluja | Interactive Portfolio

A modern, interactive portfolio website showcasing visual storytelling and creative work. Built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

- **Custom Cursor** - Animated cursor that expands on hover
- **Draggable Elements** - Interactive hero section elements that can be dragged on desktop
- **Parallax Scrolling** - Gallery items with different scroll speeds for depth effect
- **Responsive Design** - Mobile-optimized layout
- **Grain Overlay** - Subtle noise texture for visual polish
- **Multi-section Layout** - Work gallery, digital assistant, and contact sections

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/jasnoor-portfolio.git
   ```

2. Open `index.html` in your browser to view the portfolio

3. No build process required - the site uses CDN resources for Tailwind CSS and Font Awesome

## Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **Font Awesome** - Icon library
- **Vanilla JavaScript** - Interactive features
- **Google Fonts** - Inter typeface

## Customization

### Modify Content
- Update the HTML in `index.html` with your own content
- Replace placeholder images and videos
- Update social media links

### Update Styling
- Inline styles are in the `<style>` tag
- Tailwind classes are used throughout for responsive design
- CSS variables for cursor sizes can be modified in `:root`

### Add External Resources
- Place images in the `assets/` folder
- Link them in the HTML
- Consider optimizing images for web
